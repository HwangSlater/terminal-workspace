FEATURES

알림 확인부터 답장까지, 터미널 안에서

S

Slack

지켜볼 채널과 팀원만 골라 폴링합니다. /send #채널 메시지로 바로 답장하고, /로 채널을 검색합니다.

G

GitHub

선택한 저장소의 열린 PR을 알림으로 받습니다. /repo-watch owner/repo로 감시 목록을 명령줄에서 통째로 교체합니다.

C

Calendar

여러 개의 비공개 iCal 주소를 연결해 다가오는 일정을 한 패널에서 봅니다. 월 그리드 뷰(Ctrl+M)도 있습니다.

:

명령줄 하나로 전부

피커 오버레이 없이도 : 명령줄만으로 채널/저장소/캘린더 선택, 상태 변경, 알림 정리까지 조작할 수 있습니다. Tab 자동완성 지원.

◷

뽀모도로 타이머

외부 앱 없이 /pomodoro start 한 줄로 집중/휴식 사이클을 헤더에 띄워둡니다.

⌁

Local First

모든 데이터는 로컬 임베디드 저장소(redb)에 남습니다. 토큰은 OS 키체인에, 서버나 원격 계정은 필요 없습니다.

INSTALL

미리 빌드된 바이너리로 바로 시작

curl | sh 
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/HwangSlater/terminal-workspace/releases/latest/download/app-installer.sh | sh
PowerShell 
powershell -ExecutionPolicy Bypass -c "irm https://github.com/HwangSlater/terminal-workspace/releases/latest/download/app-installer.ps1 | iex"

설치 후 새 터미널에서 app을 실행하세요. Windows용 .msi 인스톨러와 소스 빌드 안내는 README에 있습니다.
