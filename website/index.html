Local First

알림·일정·설정 모두 로컬 임베디드 저장소(redb)에 남습니다. 클라우드 계정이나 원격 서버가 필요 없습니다.

Zero Configuration

설정 파일을 손으로 쓸 일이 없습니다. 처음 실행하면 config.toml과 데이터베이스가 자동으로 만들어집니다.

Cross-Platform

Windows·macOS·Linux를 동등한 1순위로 지원합니다. 토큰은 각 OS 키체인에, 알림은 데스크톱 토스트로.

FEATURES

알림 확인부터 답장까지, 터미널 안에서

S

Slack

지켜볼 채널과 팀원만 골라 폴링합니다. /send #채널 메시지로 바로 답장하고, /로 채널을 검색합니다.

G

GitHub

선택한 저장소의 열린 PR을 알림으로 받습니다. /repo-watch owner/repo로 감시 목록을 명령줄에서 통째로 교체합니다.

C

Calendar

여러 개의 비공개 iCal 주소를 연결해 다가오는 일정을 한 패널에서 봅니다. 월 그리드 뷰(Ctrl+M)도 있습니다.

:

명령줄 하나로 전부

피커 오버레이 없이도 : 명령줄만으로 채널/저장소/캘린더 선택, 상태 변경, 알림 정리까지 조작할 수 있습니다. Tab 자동완성 지원.

◷

뽀모도로 타이머

외부 앱 없이 /pomodoro start 한 줄로 집중/휴식 사이클을 헤더에 띄워둡니다.

⌁

Local First

모든 데이터는 로컬 임베디드 저장소(redb)에 남습니다. 토큰은 OS 키체인에, 서버나 원격 계정은 필요 없습니다.

SETUP

연동 하나당 1분, OAuth 없이

Slack

Ctrl+S / Ctrl+P
  1. Slack App을 만들고 Bot Token 스코프(channels:history, channels:read, users:read, chat:write — 비공개 채널은 groups:history/groups:read도)로 설치해 Bot Token(xoxb-...)을 발급받으세요.
  2. Ctrl+S로 토큰을 입력하면 바로 연결됩니다 — OS 키체인에 저장되고 설정 파일엔 절대 남지 않습니다.
  3. 메시지를 받을 채널에 봇을 초대하세요 (/invite @봇이름).
  4. Ctrl+P로 채널/팀원을 선택하세요. 목록이 길면 /로 검색합니다.
  5. 명령줄에서 바로: /send #채널 메시지, /away 등 상태 변경, /slack-watch #채널1 #채널2로 목록 통째 교체.
자세히 보기 →

GitHub

Ctrl+G / Ctrl+R
  1. GitHub → Settings → Developer settings → Personal access tokens에서 repo 스코프로 Classic PAT(ghp_...)를 발급받으세요.
  2. Ctrl+G로 토큰을 입력하면 바로 연결됩니다.
  3. Ctrl+R로 접근 가능한 저장소를 선택하면, 열린 PR이 알림 패널에 표시됩니다. /로 검색할 수 있습니다.
  4. 명령줄에서 /repo-watch owner/repo owner/repo2로 감시 목록을 통째로 교체할 수 있습니다.
자세히 보기 →

Calendar

Ctrl+L / Ctrl+K / Ctrl+M
  1. Google Calendar → 설정 → 캘린더 통합 → 비공개 iCal 형식 주소를 복사하세요. OAuth나 로그인 절차가 필요 없습니다.
  2. Ctrl+L로 이름(예: "회사")과 주소를 입력하면 바로 연결됩니다. 반복 일정도 인식합니다.
  3. 여러 캘린더를 동시에 연결할 수 있습니다 — Ctrl+L을 다시 눌러 하나씩 추가하세요.
  4. Ctrl+K로 관리·제거·이름 변경(e)을 하거나, 명령줄에서 /calendar-rename//calendar-remove로도 가능합니다.
  5. Ctrl+M으로 월 단위 그리드 뷰를 볼 수 있습니다 ([/]로 이전/다음 달).

curl | sh 
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/HwangSlater/terminal-workspace/releases/latest/download/app-installer.sh | sh
PowerShell 
powershell -ExecutionPolicy Bypass -c "irm https://github.com/HwangSlater/terminal-workspace/releases/latest/download/app-installer.ps1 | iex"
